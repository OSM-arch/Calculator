<h2>✨ Key Features</h2>
<ul>
  <li>Full keyboard support (0-9, +, -, *, /, Enter, Backspace)</li>
  <li>Mouse/touch operation</li>
  <li>Error handling for invalid inputs</li>
  <li>Responsive design with CSS Grid</li>
  <li>Visual button presses</li>
</ul>
